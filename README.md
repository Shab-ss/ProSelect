# ProSelect
Machine Learning Pipeline to select probiotic strains in cosmetic formulizations

## Project Overview
The human microbiome encompasses a complex, diverse and a heterogeneous community of microbes. The nature of these microorganisms is that they interact. The interactions are mostly among themselves and also with the host. This project determines the probiotic strains that have optimal performance in cosmetic formulations considering efficacy, regulatory standards, and skin compatibility.
